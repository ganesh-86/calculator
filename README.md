# Modern Calculator UI

A beautiful, functional, glassmorphic calculator built with React and Tailwind CSS.

## Features
- **Glassmorphic Aesthetic:** Deep slate gradients combined with blurred white-glass panels.
- **Micro-interactions:** Tactile feedback on button presses via smooth CSS transforms.
- **Robust Math Logic:** Evaluates full floating-point operations cleanly while handling edge cases like division by zero.
- **Responsive Layout:** Grid-based constraints keep the keypad crisp across different screen sizes.

## Tech Stack
- React 18
- Tailwind CSS 3
- Vite

## How to Run Locally

First, ensure you have Node.js installed, then install the local dependencies:
```bash
npm install
```

Start the Vite development server:
```bash
npm run dev
```

Build the application for production:
```bash
npm run build
```
